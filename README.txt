Database field for Moodle: Email notification
=============================================

A data field type component to moodle. Send a email notification when a record is created or updated.

Package tested in: moodle 3.11+

QUICK INSTALL
==============
Download zip package, extract the share folder and upload this folder into mod/data/field/.
